## Phase 5: 빌드 & 준비 (1일)

### 📦 5.1 프로덕션 빌드

```bash
npm run build
```

빌드 결과 확인:
- [ ] `.ait` 디렉토리 생성
- [ ] 번들 파일 존재
- [ ] 크기 확인 (< 5MB)

### 📦 5.2 메타데이터 준비

`bedrock.config.ts` 확인:

```typescript
{
  brand: {
    displayName: '복리 계산기',  // ✅ 최종 확인
    primaryColor: '#3182F6',      // ✅ 브랜드 색상
    icon: 'https://...',          // ✅ 아이콘 URL (512x512 PNG)
    bridgeColorMode: 'basic',
  },
  permissions: [],                 // ✅ 필요한 권한만
}
```

### 📦 5.3 앱 아이콘 준비

**요구사항**:
- 크기: 512x512px
- 포맷: PNG
- 배경: 불투명 (투명 배경 금지)
- 모서리: 둥글지 않음 (시스템이 자동 처리)

**업로드**:
1. 이미지 호스팅 (AWS S3, Cloudinary 등)
2. HTTPS URL 생성
3. `bedrock.config.ts`에 입력

## Phase 7: 출시 제출 (1일)

### 📤 7.1 번들 업로드

1. 콘솔 > 개발 > 배포
2. `.ait` 번들 업로드
3. 버전 정보 입력:
   - 버전: 1.0.0
   - 변경사항: 첫 출시

### 📤 7.2 앱 정보 작성

**필수 정보**:
- [ ] 앱 이름 (한글/영문)
- [ ] 한 줄 소개 (30자 이내)
- [ ] 상세 설명 (500자 이내)
- [ ] 스크린샷 (최소 3장)
  - 크기: 1080x1920px
  - 메인 화면, 결과 화면, 상세 화면
- [ ] 카테고리
- [ ] 태그 (최대 5개)

**예시**:
```
앱 이름: 복리 계산기
한 줄 소개: 투자 수익을 쉽게 계산하는 복리 계산기
설명:
원금, 이자율, 기간을 입력하면 복리 계산 결과를
연도별로 자세히 보여드려요. 월 추가 납입도 지원하고,
입력값은 자동 저장되어 편리해요.

주요 기능:
• 정확한 복리 계산
• 월 납입금 지원
• 연도별 상세 내역
• 자동 저장/복원

카테고리: 금융/재테크
태그: 복리, 계산기, 투자, 재테크, 저축
```

### 📤 7.3 테스트 계정 (필요 시)

로그인 필요한 경우:
- 테스트 계정 ID/PW 제공
- 테스트 시나리오 작성
